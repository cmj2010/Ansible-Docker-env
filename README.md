# Ansible-Docker-env
Docker for ansible enviroment to connect FortiGate  

# A few cli to play with:
#Docekr test/dev  
docker build  -t mjchenansible .

#Run  
docker run -i -t mjchenansible /bin/bash  

#To mount a local directory for playbook dev/test:
